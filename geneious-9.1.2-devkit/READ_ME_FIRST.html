Geneious Plugin Development Kit - Readme

The Geneious Public API consists of a set of classes and interfaces which allow you to write plugins for Geneious. The API is very powerful and so useful that many parts of the standard Geneious application, such as the sequence viewer and local database service, are implemented through it. For a full list of the plugins that come with Geneious click the "Tools" menu, select "Plugins", then "Customize Feature Set".

Geneious is written in version 6 of the Java programming language. If you want to write plugins for Geneious you will need some knowledge of Java, and the Java Development Kit (JDK) which can be downloaded from Sun here if you're using Windows, from Apple here if you're using Mac OS X or via your package manager or the Sun Java website if you're using Linux.

For a thorough introduction and overview of the Geneious Public API, and links to documentation for the individual classes and methods see the main page of the Public API javadoc.

For instructions on setting up your development environment, running the provided examples and building your own plugins follow the instructions in the setup file.

For a complete step-by-step guide to building a fully functioning plugin that wraps a command-line program using Eclipse have a look at the Phobos Plugin Development PDF document.
