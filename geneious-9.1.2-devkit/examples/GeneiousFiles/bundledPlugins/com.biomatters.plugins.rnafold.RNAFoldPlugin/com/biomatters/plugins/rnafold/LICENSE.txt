The RNAfold and RNAcofold executable files are built from source code
in the Vienna RNA Package downloaded from
http://www.tbi.univie.ac.at/RNA/

See source code package files for copyright notices and licensing terms
for the Vienna RNA Package and third party library software that is
compiled with it to build the executable files.

given to the authors and the Institute for Theoretical Chemistry of the

The complete source package contains software whose copyrights are owned by
